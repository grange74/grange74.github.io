Terraform Hello World on AWS

May 20, 2015 · 3 minute read

So you’ve heard about Terraform, you here that its another cool product developed by HashiCorp and so you want it try it out.

Here’s one of the simplest tutorials possible.

provider "aws" {
    access_key = "ABC123"
    secret_key = "DEF456"
    region = "us-east-1"
}

resource "aws_instance" "web" {
    ami = "ami-1ecae776"
    instance_type = "t2.micro"
}

$ terraform plan
Refreshing Terraform state prior to plan...
